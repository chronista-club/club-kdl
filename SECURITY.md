# Security Policy

## Supported Versions

最新の minor version のみセキュリティ修正の対象です。

| Version | Supported |
|---------|-----------|
| 0.5.x   | ✅        |
| < 0.5   | ❌        |

## 脆弱性の報告

**GitHub Issue では報告しないでください。**

代わりに以下のいずれかの手段でご連絡ください:

1. **GitHub Security Advisory** (推奨)
   - [Report a vulnerability](https://github.com/chronista-club/club-kdl/security/advisories/new) からプライベートに報告
2. メール
   - `mito@chronista.club` へ「club-kdl security」を件名に含めて送信

## 期待する対応時間

- **初回応答**: 7 営業日以内
- **修正リリース** (深刻度に応じる):
  - Critical: 14 日以内
  - High: 30 日以内
  - Medium / Low: 次の minor release で対応

## 報告に含めてほしい情報

- 影響範囲 (どのバージョンか / どの機能か)
- 再現手順
- 想定される影響
- 可能であれば修正案

ご協力ありがとうございます。
