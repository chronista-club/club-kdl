# Contributor Covenant Code of Conduct

## 私たちの誓約

私たちメンバー、貢献者、リーダーは、年齢、体格、目に見えるまたは見えない障害、民族性、性的特徴、性自認と性表現、経験のレベル、教育、社会経済的地位、国籍、容姿、人種、宗教、または性的アイデンティティと指向に関わらず、私たちのコミュニティへの参加が誰にとってもハラスメントのない体験となるように努めます。

## 私たちの基準

ポジティブな環境に貢献する行動の例:

- 他人への共感と優しさを示す
- 異なる意見、視点、経験を尊重する
- 建設的なフィードバックを与え、また受け入れる
- 過ちの責任を受け入れ、影響を受けた人々に謝罪し、その経験から学ぶ
- 個人ではなく、コミュニティ全体にとって最善のことに焦点を当てる

容認できない行動の例:

- 性的な言葉や画像の使用、あらゆる種類の性的注目や誘い
- 荒らし行為、侮辱や軽蔑的なコメント、個人的または政治的な攻撃
- 公的または私的なハラスメント
- 物理的または電子メールアドレスなどの他人の個人情報を、明示的な許可なしに公開すること
- 専門的環境で合理的に不適切と見なされる可能性のあるその他の行為

## 執行責任

コミュニティリーダーは、許容される行動の基準を明確にし、執行する責任を負い、不適切、脅迫的、攻撃的、有害と見なされるあらゆる行動に対して、適切で公正な是正措置を講じます。

## 適用範囲

この行動規範は、すべてのコミュニティスペースに適用され、また、個人が公にコミュニティを代表しているときにも適用されます。

## 執行

虐待的、嫌がらせ的、またはその他の容認できない行動の事例は、コミュニティリーダー (`mito@chronista.club`) に報告できます。すべての苦情は迅速かつ公正に確認、調査されます。

## 帰属

この行動規範は、[Contributor Covenant](https://www.contributor-covenant.org) version 2.1 を翻案したものです。
